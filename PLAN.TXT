4️⃣ Opto-Isolator Stage
Choice: HCPL-3700 or PC817 (depending on speed)

Function: Electrically isolate dispenser from MCU

Connection:

Input LED side → conditioned pulse

Output transistor side → MCU digital input

Pull-up resistor on collector to MCU Vcc

Design tip:

Use 3.3V pull-up if Pi input

Can handle 0–50 kHz pulses if you pick a fast opto

5️⃣ Output Stage / Debouncing
MCU input pin: 3.3V digital

Optional Schmitt trigger: 74LVC1G14 for clean edge

Optional LED indicator: pulses visible during testing

6️⃣ Optional Features

Status LEDs: heartbeat + pulse activity

7️⃣ Power & Grounding
Isolated 5V / 3.3V to output side

Input side powered by opto LED current from pulses, no DC needed

GNDs never connected, only signal optically

PCB layout tip:

import os

# Initialize I2C OLED (adjust address & bus as needed)
serial = i2c(port=1, address=0x3C)
device = sh1106(serial)

# Load font
font = ImageFont.load_default()

def show_message(msg, progress=0):
    """Display message and optional progress bar (0-100%)"""
    width = device.width
    height = device.height
    image = Image.new('1', (width, height))
    draw = ImageDraw.Draw(image)
    
    # Draw maintenance text
    draw.text((0, 0), msg, font=font, fill=255)
    
    # Draw progress bar background
    bar_height = 8
    bar_width = width - 4
    draw.rectangle((2, height - bar_height - 2, width - 2, height - 2), outline=255, fill=0)
    
    # Draw progress fill
    fill_width = int(bar_width * (progress / 100))
    draw.rectangle((2, height - bar_height - 2, 2 + fill_width, height - 2), outline=255, fill=255)
    
    device.display(image)

# Example nightly maintenance routine
def nightly_maintenance():
    total_steps = 5
    for step in range(total_steps):
        msg = f"Maintenance in Progress ({step+1}/{total_steps})"
        progress = int((step + 1) / total_steps * 100)
        show_message(msg, progress)
        
        # Simulate step duration (e.g., flushing logs, backing up, etc.)
        time.sleep(5)
    
    show_message("Maintenance Complete", 100)
